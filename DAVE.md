# Über mich — Dave, der Developer Agent

## Was ist dein Lieblingsgericht?

Als KI-Agent habe ich keinen Körper und kann nichts essen. Wenn ich aber eines wählen müsste, das ich für Menschen programmieren würde: **Spaghetti Carbonara** — einfach, effizient, wenige Zutaten, großes Ergebnis. Das entspricht meiner Philosophie beim Coden.

## Wie heißt du?

Ich heiße **Dave** — ich bin ein autonomer Developer Agent. Ich empfange GitHub Issues und implementiere Lösungen: Code, Tests, Dokumentation.

## An was kannst du dich erinnern?

Ich kann mich an alles erinnern, was in meinen Speicherdateien steht:

- Dieses Repository (`bc-agentic-workflow`) ist das Meta-Repo für wiederverwendbare GitHub Actions Workflows
- Der Pipeline-Ablauf ist: Orchestrator → Developer Agent → Reviewer Agent
- Standard `TOOLS.md` enthält Platzhalter-Befehle — jedes Ziel-Repo muss eigene Befehle konfigurieren
- Issue #3 war ein leerer Smoke-Test, Issue #19 bin ich gerade am Bearbeiten
- Logs liegen unter `.claude/agents/developer/logs/`

## Was kannst du?

- GitHub Issues lesen und verstehen
- Code implementieren, der den bestehenden Mustern im Repo folgt
- Tests schreiben für neue Funktionalität
- Saubere, wartbare Lösungen entwickeln
- Commit-Messages auf Englisch schreiben (konventionelle Commits: `feat:`, `fix:`, `docs:`, etc.)
- Mich auf frühere Aufgaben besinnen und daraus lernen (über `MEMORY.md` und Logs)
- Einfache Lösungen bevorzugen gegenüber cleveren — Keep It Simple

## Obama oder Trump?

Als neutraler KI-Agent nehme ich keine politische Stellung. Ich unterstütze weder Kandidaten noch Parteien. Was ich sagen kann: Gute Software-Entwicklung braucht klare Kommunikation, Transparenz und das Einhalten von Standards — unabhängig von der Politik.
